# Космическая туризм

Учебный проект на Java с Hibernate ORM, демонстрирующий работу с базой данных через консольный интерфейс.

## Технологии

- Java 17+
- Hibernate 6.x (JPA 3.x)
- PostgreSQL
- Maven

## Возможности

- Полный CRUD для 9 сущностей: рейсы, корабли, капсулы, астронавты, билеты, еда, тесты, результаты тестов, багаж
- 10 бизнес-запросов на JPQL с агрегацией, подзапросами и навигацией по связям
- Автоматическое создание и заполнение базы данных при первом запуске
- Удобный консольный интерфейс с меню и таблицами

## Структура проекта

```
src/main/java/com/flight/
├── Main.java                    # Точка входа
├── entity/                      # JPA-сущности (9 классов)
├── repository/                  # Репозитории (GenericRepository + 9 наследников)
├── service/
│   └── BusinessQueryService.java  # 10 JPQL-запросов
├── console/
│   └── ConsoleApp.java          # Консольный интерфейс
└── util/
    ├── HibernateUtil.java       # Фабрика EntityManager
    └── DataSeeder.java          # Сидер начальных данных
```

## Структура базы данных

Проект включает 9 связанных таблиц:

- **flights** — рейсы (направление, даты вылета и прилёта)
- **spaceships** — корабли, привязанные к рейсам, с указанием количества капсул
- **capsules** — капсулы двух классов: эконом и панорамный люкс
- **astrounauts** — астронавты с ролью, весом и ростом
- **tickets** — билеты, связывающие астронавта и рейс с ценой
- **foods** — запасы еды на каждом корабле
- **tests** — медицинские тесты с уникальной темой
- **results** — результаты тестов астронавтов (составной первичный ключ)
- **bags** — багаж астронавтов с весом

## Консольный интерфейс

После запуска отображается главное меню с разделами для каждой сущности и бизнес-запросами. Каждый раздел предоставляет операции: просмотр всех записей, поиск по ID и критериям, создание, обновление и удаление. Раздел бизнес-запросов включает: выручку по направлениям, загрузку капсул, топ астронавтов, популярность классов капсул, средний вес багажа по ролям, астронавтов без билетов, свободные капсулы на рейс и статистику по тестам.
